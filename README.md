# CodeAlpha_CreditScoringModel

## 📌 Overview

This project is a Machine Learning based Credit Scoring Model developed during the CodeAlpha Internship Program. The model predicts an individual's creditworthiness using financial and historical data.

## 🚀 Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature engineering
* Creditworthiness prediction
* Model evaluation using multiple metrics

## 🤖 Algorithms Used

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## 📊 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* ROC-AUC

## 📂 Dataset Features

* Income
* Debts
* Payment History
* Credit Utilization
* Financial Records

## ▶️ Project Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Prediction

## 🎯 Objective

To build a Machine Learning model capable of predicting whether an individual is creditworthy based on financial history and related attributes.
